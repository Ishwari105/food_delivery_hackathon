# Food Delivery Data Integration Hackathon

## Description
This project combines transactional, user, and restaurant data from CSV, JSON, and SQL formats into a unified dataset.

## Files Included
- food_delivery_analysis.ipynb
- final_food_delivery_dataset.csv

## Key Steps
- Loaded CSV, JSON, SQL data
- Performed left joins
- Created final dataset for analysis

## Tools Used
- Python
- Pandas
- SQLite
- Jupyter Notebook
